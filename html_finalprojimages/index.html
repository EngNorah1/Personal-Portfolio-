Real-Time Weather Connector

This project develops a communication system between a server and a client using the Transmission Control Protocol (TCP) in C#. The server retrieves weather data from the OpenWeather API and sends it to the client once a connection is established. The client then displays the received weather data, which includes the city name, weather conditions, temperature, humidity, and wind speed. This project provides a simple yet effective model for establishing a reliable and secure TCP connection between a server and a client, demonstrating core concepts of network programming.
